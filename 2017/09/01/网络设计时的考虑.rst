网络设计时的考虑
*****************



.. author:: default
.. categories:: none
.. tags:: Network, Program
.. comments::

设计一个游戏网络框架时需要考虑的问题：

服务端部分

*   新连接进入事件，创建一个连接，放入连接管理
*   读取连接数据，将由其它部分处理
*   连接断开事件
*   连接活性检测
*   协议版本检测
*   加密方法协商
*   序列化与反序列化

客户端

*   主动连接服务端
*   连接建立后：

    *   验证协议版本
    *   协商加密方法

*   序列化与反序列化
*   接收网络数据处理
